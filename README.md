# h5-nav

#### 介绍
手机导航站

#### 软件架构
- 简洁极速的导航页，可代替手机浏览器的首页，使用Service Worker技术，支持页面离线访问
- 可作为WebAPP添加到手机桌面，chrome浏览器可以读取设置的图标，UC暂只能使用默认图标



#### 使用说明

1. 手机浏览器中打开：https://niefy.gitee.io/h5-nav
2. 复制链接，设置为浏览器首页
3. 无法设置首页的浏览器，可以添加到桌面，以后用桌面快捷入口进入浏览器
